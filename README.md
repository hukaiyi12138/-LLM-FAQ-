# -LLM-FAQ-
项目背景：\
在客服服务场景中，积累了大量历史客服问答数据，但这些数据多为非结构化形式，缺乏系统整理与归类，导致相似问题重复处理、客服响应效率低，同时难以快速挖掘高频问题、形成标准化服务体系，无法满足高效客服服务与知识沉淀的需求。

任务：\
针对上述痛点，核心任务是搭建高效FAQ构建流程，完成历史问答数据结构化整理、相似问题归并与标准问答生成，形成可复用FAQ知识库，支持相似问题检索与高频问题挖掘。

解决方案：\
1.相似问题归并：采用 HNSW 进行高维向量近邻检索，结合 SinglePass 实现流式聚类，将相似问题归并为相同问题簇。\
2.标准化FAQ生成：基于 LLM 为每个问题簇生成代表问题与标准回答，形成结构化 FAQ 知识库。

技术成果：\
1.实现核心功能落地：完成FAQ构建流程搭建与结构化知识库生成，支持相似问题检索和高频问题挖掘，提升客服效率、支撑服务优化。\
2.完成技术成果沉淀：项目代码开源至GitHub（链接：），实现技术复用，为同类项目提供参考，达成核心目标。
